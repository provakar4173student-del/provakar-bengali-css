# PROVAKAR - ওয়েবসাইট কীভাবে আপনার পিসিতে নামাবেন

## পদ্ধতি ১: এই ওয়ার্কস্পেস থেকে সরাসরি (Direct from Workspace)

### ধাপ ১: সব ফাইল সিলেক্ট করুন
আমি আপনাকে সব ফাইল একটি জিপ ফাইল করে দিচ্ছি। নিচের কমান্ডটি রান করুন:

```bash
# টার্মিনালে এই কমান্ডটি লিখুন (আমি একটি জিপ ফাইল তৈরি করে দেব)
```

### ধাপ ২: জিপ ফাইল ডাউনলোড করুন
আমি একটি জিপ ফাইল তৈরি করে দিচ্ছি যা আপনি ডাউনলোড করতে পারবেন।

---

## পদ্ধতি ২: GitHub এ আপলোড করে নামান (সবচেয়ে ভালো পদ্ধতি)

### ধাপ ১: GitHub এ রিপোজিটরি তৈরি করুন
১. https://github.com এ যান
২. লগইন করুন
৩. "New repository" এ ক্লিক করুন
৪. নাম দিন: `provakar-bengali`
৫. "Create repository" এ ক্লিক করুন

### ধাপ ২: ফাইলগুলো আপলোড করুন
১. "Add file" → "Upload files" এ ক্লিক করুন
২. নিচের সব ফাইল ও ফোল্ডার আপলোড করুন:

```
provakar-bengali/
├── index.html
├── chapters.html
├── chapter.html
├── quiz.html
├── dashboard.html
├── mock-test.html
├── settings.html
├── css/ (পুরো ফোল্ডার)
├── js/ (পুরো ফোল্ডার)
└── *.md (সব ডকুমেন্টেশন ফাইল)
```

### ধাপ ৩: আপনার পিসিতে ক্লোন করুন
```bash
git clone https://github.com/আপনার-ইউজারনেম/provakar-bengali.git
```

## 🔧 কোড এরর চেক ও ফিক্স

আমি নিচের এররগুলো ঠিক করছি:

### ✅ ঠিক করা হয়েছে:
১. `js/app.js` এ স্পেলিং মিসটেক ("операция" → "অপারেশন")
২. `chapter.html` এ স্ক্রিপ্ট লোডিং অর্ডার ঠিক করা হয়েছে
৩. `quiz.html` এ স্ক্রিপ্ট লোডিং ঠিক করা হয়েছে

# 🚀 PROVAKAR - GitHub Pages এ হোস্ট করার সম্পূর্ণ গাইড

## পদ্ধতি ১: GitHub ওয়েব ইন্টারফেস ব্যবহার করে (সবচেয়ে সহজ)

### ধাপ ১: GitHub একাউন্ট তৈরি করুন
1. https://github.com এ যান
2. "Sign up" এ ক্লিক করুন
3. ইউজারনেম, ইমেইল ও পাসওয়ার্ড দিন
4. ভেরিফিকেশন করুন

### ধাপ ২: নতুন রিপোজিটরি তৈরি করুন
1. লগইন করার পর ডান দিকে "+" বাটনে ক্লিক করুন
2. "New repository" সিলেক্ট করুন
3. রিপোজিটরির নাম দিন: `provakar-bengali`
4. "Public" সিলেক্ট করুন
5. "Add a README file" চেকবক্সে টিক দিন
6. "Create repository" এ ক্লিক করুন

### ধাপ ৩: ফাইলগুলো আপলোড করুন
1. রিপোজিটরি ওপেন হলে "Add file" → "Upload files" এ ক্লিক করুন
2. আপনার কম্পিউটার থেকে নিচের সব ফাইল সিলেক্ট করে আপলোড করুন:

```
provakar-bengali/
├── index.html
├── chapters.html
├── chapter.html
├── quiz.html
├── dashboard.html
├── mock-test.html
├── settings.html
├── css/
│   ├── style.css
│   ├── animations.css
│   ├── responsive.css
│   ├── chapter.css
│   ├── quiz.css
│   ├── dashboard.css
│   └── mock-test.css
├── js/
│   ├── storage.js
│   ├── app.js
│   ├── search.js
│   ├── settings.js
│   ├── chapters-data.js
│   ├── chapter-view.js
│   ├── quiz.js
│   └── dashboard.js
├── README.md
├── DELIVERY-SUMMARY.md
└── HOW-TO-ADD-CHAPTERS.md
```

3. "Commit changes" বাটনে ক্লিক করুন

### ধাপ ৪: GitHub Pages চালু করুন
1. রিপোজিটরির "Settings" ট্যাব এ যান
2. বাম মেনুতে "Pages" (বা "Pages and deployment") সিলেক্ট করুন
3. "Build and deployment" এর নিচে "Branch" সিলেক্ট করুন → `main`
4. "Save" এ ক্লিক করুন
5. ২-৩ মিনিট অপেক্ষা করুন

### ধাপ ৫: ওয়েবসাইট দেখুন
আপনার ওয়েবসাইট লাইভ হবে এই ঠিকানায়:
```
https://আপনার-ইউজারনেম.github.io/provakar-bengali/
```

## পদ্ধতি ২: Git কমান্ড লাইন ব্যবহার করে

### ধাপ ১: Git ইনস্টল করুন
- https://git-scm.com থেকে Git ডাউনলোড ও ইনস্টল করুন

### ধাপ ২: প্রোজেক্ট ইনিশিয়ালাইজ করুন
```bash
cd provakar-bengali
git init
git add .
git commit -m "Initial commit: PROVAKAR Bengali Exam Prep Website"
```

### ধাপ ৩: GitHub রিপোজিটরির সাথে যুক্ত করুন
```bash
git remote add origin https://github.com/আপনার-ইউজারনেম/provakar-bengali.git
git branch -M main
git push -u origin main
```

### ধাপ ৪: GitHub Pages চালু করুন

## 🔧 সমস্যা সমাধান (Troubleshooting)

### সমস্যা ১: ওয়েবসাইট লোড হচ্ছে না
**সমাধান:**
- রিপোজিটরির নাম এবং URL মিল আছে কিনা চেক করুন
- `index.html` ফাইল রুট ডিরেক্টরিতে আছে কিনা চেক করুন

### সমস্যা ২: CSS বা JS লোড হচ্ছে না
**সমাধান:**
- সব ফাইল ঠিকমতো আপলোড হয়েছে কিনা চেক করুন
- ফাইল পাথ ঠিক আছে কিনা চেক করুন (কেস সেনসিটিভ)

### সমস্যা ৩: বাংলা ফন্ট দেখা যাচ্ছে না
**সমাধান:**
- ইন্টারনেট কানেকশন আছে কিনা চেক করুন (Google Fonts CDN এর জন্য)
- ব্রাউজার রিফ্রেশ করুন

## 🎨 কাস্টম ডোমেইন যোগ করতে চান?

আপনি যদি নিজের ডোমেইন ব্যবহার করতে চান:
1. একটি ডোমেইন কিনুন (যেমন: provakar.com)
2. GitHub Pages এর "Custom domain" এ ডোমেইন নাম লিখুন
3. DNS সেটিংস কনফিগার করুন

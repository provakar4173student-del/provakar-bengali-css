# PROVAKAR - বাংলা পরীক্ষা প্রস্তুতি ওয়েবসাইট

## 📋 বিবরণ (Description)

PROVAKAR হলো পশ্চিমবঙ্গ গ্রাম পঞ্চায়েত পরীক্ষার বাংলা বিষয়ের সম্পূর্ণ প্রস্তুতির জন্য তৈরি একটি ওয়েবসাইট। এই ওয়েবসাইটে রয়েছে:
- ১৭টি অধ্যায়ের বিস্তারিত পড়াশোনা
- শর্টকাট মেথড ও ট্রিকস
- MCQ কুইজ (৩টি মোড)
- মক টেস্ট
- প্রগতি ট্র্যাকিং

## 📝 নতুন MCQ প্রশ্ন কীভাবে যোগ করবেন (How to Add Questions)

### পদ্ধতি ১: সরাসরি কোড এডিট করে (Direct Code Edit)

`js/chapters-data.js` ফাইল খুলুন। প্রতিটি অধ্যায়ের জন্য `questions` অ্যারেতে নতুন প্রশ্ন যোগ করুন:

```javascript
questions: [
    {
        question: "আপনার প্রশ্ন এখানে লিখুন?",
        options: [
            "অপশন A",
            "অপশন B",
            "অপশন C",
            "অপশন D"
        ],
        correct: 0,  // সঠিক উত্তরের ইনডেক্স (0-3)
        explanation: "সঠিক উত্তরের ব্যাখ্যা এখানে লিখুন।",
        trick: "মনে রাখার ট্রিক বা শর্টকাট।",
        formula: "প্রযোজ্য হলে সূত্র।"
    },
    // আরও প্রশ্ন যোগ করুন...
]
```

### পদ্ধতি ২: JSON ফরম্যাটে ইমপোর্ট করে (JSON Import)

ওয়েবসাইটের সেটিংস পেজে "ডেটা ইমপোর্ট" অপশন ব্যবহার করুন:

```json
{
  "chapter": 1,
  "questions": [
    {
      "question": "ধ্বনি ও বর্ণের মধ্যে প্রধান পার্থক্য কী?",
      "options": [
        "ধ্বনি লেখা যায়, বর্ণ শোনা যায়",
        "ধ্বনি শোনা যায়, বর্ণ লেখা যায়",
        "দুটি একই",
        "কোনোটিই নয়"
      ],
      "correct": 1,
      "explanation": "ধ্বনি হলো মুখ থেকে উচ্চারিত শব্দাংশ যা শোনা যায়...",
      "trick": "ধ্বনি = ধ্বনিত = শোনা | বর্ণ = বর্ণনা = লেখা"
    }
  ]
}
```

## 🌐 GitHub Pages এ কীভাবে হোস্ট করবেন (How to Deploy to GitHub Pages)

### ধাপ ১: GitHub এ রিপোজিটরি তৈরি করুন
1. GitHub এ লগইন করুন
2. "New repository" এ ক্লিক করুন
3. রিপোজিটরির নাম দিন: `provakar-bengali`
4. "Public" সিলেক্ট করুন
5. "Create repository" এ ক্লিক করুন

### ধাপ ২: কোড আপলোড করুন
**অপশন A: GitHub ওয়েব ইন্টারফেস ব্যবহার করে**
1. রিপোজিটরিতে "Add file" → "Upload files" এ ক্লিক করুন
2. সমস্ত ফাইল আপলোড করুন

**অপশন B: Git কমান্ড ব্যবহার করে**
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/আপনার-ইউজারনেম/provakar-bengali.git
git push -u origin main
```

### ধাপ ৩: GitHub Pages চালু করুন
1. রিপোজিটরির "Settings" এ যান
2. বাম মেনু থেকে "Pages" সিলেক্ট করুন
3. "Source" এ "Deploy from a branch" সিলেক্ট করুন
4. "Branch" এ `main` সিলেক্ট করুন
5. "Save" এ ক্লিক করুন

### ধাপ ৪: ওয়েবসাইট দেখুন
কয়েক মিনিট অপেক্ষা করুন। আপনার ওয়েবসাইট লাইভ হবে:
`https://আপনার-ইউজারনেম.github.io/provakar-bengali/`
